搭建并使用Hugo博客

2018-04-05

安装指南

hugo官方网址 1. 在Hugo Release下载exe文件 2. 添加环境变量,在PATH中添加hugo文件所在的路径 3. 在dos界面中输入hugo version查看是否配置成功

生成本地静态网站

生成站点

hugo new site [填入文件夹地址 blog\site]

创建成功后生成下列文件夹 archetypes config.toml –配置文件 content –存放文章 data layouts static themes –存放主题

运行Hugo Server

hugo server –theme=hyde Hugo内置http server，在你的站点根目录执行hugo server命令，即可在浏览器里输入： http://localhost:1313 访问网站。 注明：需要指定主题，如果在config.toml中配置指定主题可以省略–theme参数

生成文章

hugo new post/搭建并使用Hugo博客.md 在content/post 目录下面会生成 test.md文件,打开文件

---
title: "搭建并使用Hugo博客"
date: 2018-04-05T12:06:42+08:00
draft: false 
---

添加博客内容

draft默认为true,改为 false才会生成静态页面

获取主题

官方主题 官方主题github

将下载来的模版放入到themes文件夹中

关联github地址

hugo –theme=hyde –baseUrl=“https://epawm7.github.io/"

关联后在public目录下生成静态页面,上传public文件夹内的文件到github即可.
